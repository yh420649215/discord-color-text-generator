How Discord ANSI Color Text Works

ANSI (American National Standards Institute) escape codes control text formatting in terminals — colors, bold, underline. Discord adapted a subset for ```ansi code blocks. To write in color on Discord, wrap your message in an ANSI code block and prefix each line with a code like [31m] for red or [32m] for green. Discord renders each line in the specified color. Using a color text generator for Discord makes this process instant — just type, style, copy, and paste.

Discord ANSI Color Code List — Supported Colors & Styles

Discord supports 8 foreground colors for text: Gray [30m], Red [31m], Green [32m], Yellow [33m], Blue [34m], Magenta [35m], Cyan [36m], White [37m]. Plus 8 background colors ([40m]–[47m]), bold text via [1m], and underlined text via [4m]. Combine codes with semicolons — for example [1;4;31;44m] makes bold, underlined, red text on a blue background. The reset code [0m] clears all active styles. What's the Discord ANSI character limit?
Discord messages are capped at 2,000 characters total, including ANSI codes. Each color code like [31m] uses 4 characters. The ```ansi wrapper adds about 10 characters. Plan for roughly 1,900 usable text characters after code overhead.
Can I use ANSI colors in Discord server names or channel topics?
No. ANSI coloring only works inside ```ansi fenced code blocks within regular chat messages. It does not work in server names, channel topics, usernames, or message embeds.
